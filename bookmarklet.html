Gympass / Wellhub v2

Extrai receita e check-ins de todas as unidades do login atual automaticamente. Percorre o dropdown de unidades, coleta dados de Finanças de cada uma e envia tudo pro Firebase.

Extrair Gympass

Como usar:

  1. Arraste o botão verde para a barra de favoritos do Chrome
  2. Acesse partners.gympass.com e faça login
  3. Vá até Finanças (menu lateral)
  4. Clique no bookmarklet — ele percorre todas as unidades do dropdown
  5. Repita para cada conta (são 3 logins: Berrini/Itaim/VO, Pinheiros/Alphaville, Moema)

⚠ Não mude de aba enquanto o bookmarklet estiver rodando!

Totalpass v2

Extrair Totalpass

Como usar:

  1. Arraste o botão verde para a barra de favoritos
  2. Acesse booking.totalpass.com e faça login

  4. Aguarde — ele percorre todas as unidades e envia pro Firebase
