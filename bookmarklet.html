Gympass / Wellhub v3

Extrai receita E check-ins de todas as unidades. Funciona em 2 fases automáticas: primeiro coleta receita (Finanças), depois coleta check-ins (Check-ins). Permite escolher o mês.

Extrair Gympass

Como usar:

  1. Arraste o botão verde para a barra de favoritos do Chrome
  2. Acesse partners.gympass.com e faça login
  3. Vá até Finanças ou Check-ins (menu lateral)
  4. Clique no bookmarklet — ele vai pedir o mês desejado
  5. Fase 1: coleta receita de todas as unidades
  6. Fase 2: redireciona para Check-ins e clique o bookmarklet de novo
  7. Repita para cada conta (3 logins)

⚠ Não mude de aba enquanto o bookmarklet estiver rodando!

Totalpass v3

Extrai receita, check-ins (visitas) e pessoas de todas as unidades do Dashboard Totalpass. Permite escolher o mês.

Extrair Totalpass

Como usar:

  1. Arraste o botão verde para a barra de favoritos
  2. Acesse booking.totalpass.com e faça login
  3. Na página do Dashboard / Visão Geral, clique no bookmarklet
  4. Ele vai pedir o mês e percorrer todas as unidades automaticamente

WhatsApp - Planos v2

Extrai os planos vendidos do grupo "Planos BLACKSHEEP" no WhatsApp Web. Permite escolher o mês.

Extrair Planos

Como usar:

  1. Arraste o botão verde para a barra de favoritos
  2. Abra web.whatsapp.com
  3. Abra o grupo "Planos BLACKSHEEP"
  4. Clique no bookmarklet — ele extrai o texto visível e envia pra parsing por IA

⚠ Para pegar mensagens antigas: abra a busca do grupo (lupa), pesquise "mensal", e depois rode o bookmarklet
